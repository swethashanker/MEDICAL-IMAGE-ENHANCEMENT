# 🏥 Medical Image Enhancement using MATLAB

## 📌 Overview

This project focuses on enhancing medical images using MATLAB image processing techniques. It improves image quality by reducing noise, enhancing contrast, and sharpening important features—making it useful for better visualization and analysis.

---

## ✨ Features

* 📂 Load medical images dynamically
* 🎨 Automatic grayscale conversion
* 🌫️ Noise reduction using Gaussian filtering
* 📊 Histogram Equalization for contrast improvement
* 🔍 CLAHE (Adaptive Histogram Equalization) for advanced enhancement
* ✨ Image sharpening for better clarity
* 📈 Histogram comparison visualization
* 🖼️ Side-by-side comparison of processing stages

---

## 🛠️ Technologies Used

* MATLAB
* Image Processing Toolbox

---

## ⚙️ How It Works

1. User selects a medical image file.
2. Image is converted to grayscale (if needed).
3. Noise is reduced using a Gaussian filter.
4. Histogram Equalization enhances contrast.
5. CLAHE further improves local contrast.
6. Image is sharpened for better edge definition.
7. Histograms are displayed for comparison.
8. Final output shows:

   * Original Image
   * Enhanced Image
   * Sharpened Image

---

## 📷 Output Preview

The program generates:

* Histogram comparison window
* Final comparison image showing all stages side-by-side

---

## 🚀 How to Run

1. Open MATLAB
2. Copy and paste the script into a `.m` file
3. Run the script
4. Select a medical image when prompted

---

## 📁 File Requirement

* Input image format: `.png` (recommended: medical scans like MRI, CT, X-ray)

---

## 📊 Processing Pipeline

```
Input Image 
   ↓
Grayscale Conversion
   ↓
Gaussian Filtering
   ↓
Histogram Equalization
   ↓
CLAHE Enhancement
   ↓
Image Sharpening
   ↓
Final Comparison Output
```
